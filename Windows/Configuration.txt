
平时最最常用快捷键:
win : 开始菜单(全局搜索)
win+e : 资源管理器
alt+tab : 切换应用程序
左键+delete : 删除
alt+f4 : 关闭当前
ctrl+shift+esc : 任务管理器,再alt+e结束进程.
win+up : 最大化

另外再补充点chrome快捷键,只用键盘单手上网是很优雅的 :)
ctrl+t : 新建标签
ctrl+w : 关闭当前标签
ctrl+数字 : 切换到从左数第几个标签
ctrl+tab : 切换到右边的标签
ctrl+shift+tab : 切换到左边的标签
alt+d : 吧焦点切换到地址栏
ctrl+f : 页面内搜索,esc取消
敲入加入了搜索引擎的网站+tab,可直接在地址栏搜索
比如我输入ta, 自动补全为taobao.com,然后桥下tab,接着搜我想要的东西,比如"键盘",然后回车.不用再打开花花绿绿的首页了

Windows 7 已经是相当优秀的操作系统了, 我觉得额外的软件必要装的不超过3个(我建议everything和dropbox), 用原生功能就可以达到很高的效率了. 归根到底不是系统本身, 而是使用者的能力. 请问自己如下几个问题:
1. 标签化管理 - 你还停留在把照片按事件或把音乐按歌手分文件夹的阶段吗? 
2. 快捷方式和快捷键 - 常用的四五十个文件夹和程序, 你能1秒之内进入吗?
3. 文件管理的方法 - 是否云备份了? 是否存在一个文件有多个版本, 急用的时候搞不清头绪?文件按类型, 项目, 时间分类, 是否有个统一但又灵活的标准? 要你找五年前某门课的实验报告你是否找得到? 要你找大概两年前某个同事发给你的某封邮件是否找得到? 要你找在网上偶然看到过又保存过的一个搞笑图片你是否找得到?
思维方式是道, 操作系统是术. 思维水平到了那个境界, 操作系统自然用得四两拨千斤, 比其他人优雅
